Description

In this stage, you will write a program that reads the name and a surname from the input and generate a correctly-sized signature.

The signature should contain the whole name and the whole surname, just one whitespace between the name and surname, and one whitespace at the beginning and at the end of the signature, but that is all. 

For the borders use the asterisk symbol: '*'.

Output examples

If the user input  Albert Einstein, the output should be:

*******************
* Albert Einstein *
*******************

If the user input Nikola Tesla, the output should be:

****************
* Nikola Tesla *
****************
